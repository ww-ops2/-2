- **行内按钮：** 每行末尾增加绿色"+"按钮
- **悬停效果：** 按钮上浮+阴影增强

**布局示例：**

---

### 4️⃣ **修复类别模态框逻辑** ✅
**问题：** 点击"取消"会关闭整个采购单对话框  
**原因：** 全局关闭按钮事件处理不当  
**解决方案：**
1. 修改 `openCategoryModal()` 函数，添加 `event.stopPropagation()`
2. 修改全局关闭按钮逻辑，只关闭当前模态框
3. 使用 `closest()` 方法精确定位所属模态框

**代码改动：**
```javascript
// app.js - 修复前
btn.addEventListener('click', () => {
  closeModal(); //  关闭所有模态框
});

// app.js - 修复后
btn.addEventListener('click', (e) => {
  e.stopPropagation(); // ✅ 阻止冒泡
  const modal = btn.closest('.modal-content').parentElement;
  modal.classList.remove('show'); // ✅ 只关闭当前
});
```
