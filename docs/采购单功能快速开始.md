## 📖 5分钟快速上手

### 第一步：打开采购单管理

1. 登录后，点击左侧菜单 **"采购单管理"**
2. 点击右上角 **"+ 新建采购单"** 按钮

### 第二步：添加第一个物品

在弹出的模态框中：

```
基本信息：
├─ 采购日期：2026-06-09（默认今天）
├─ 采购人：张三
└─ 供应商：XX公司

点击 "+ 添加物品" 按钮，填写第一行：
├─ 类别：输入 "办公用品"（会自动创建）
├─ 物品名称：中性笔
├─ 商品编码：SKU00001（自动生成，只读）
├─ 品牌：晨光
├─ 型号：0.5mm
├─ 数量：100
├─ 单位：盒
└─ 单价：15.00
    └─ 金额：1500.00（自动计算）
```

### 第三步：添加第二个物品

再次点击 **"+ 添加物品"**，填写第二行：

```
├─ 类别：选择 "办公用品"（从下拉列表）
├─ 物品名称：A4打印纸
├─ 商品编码：SKU00002
├─ 品牌：得力
├─ 型号：70g
├─ 数量：50
├─ 单位：包
└─ 单价：25.00
    └─ 金额：1250.00
```

### 第四步：体验智能提示

添加第三个物品，测试历史记录：

```
├─ 类别：选择 "办公用品"
├─ 物品名称：中性笔（与第一个物品相同）
├─ 商品编码：SKU00003
├─ 品牌：点击输入框 → 下拉显示 "晨光" ✨
├─ 型号：点击输入框 → 下拉显示 "0.5mm" ✨
├─ 数量：50
├─ 单位：盒
└─ 单价：15.00
    └─ 金额：750.00

合计：3500.00元
```

### 第五步：提交采购单

1. 检查所有信息无误
2. 填写备注（可选）
3. 点击 **"提交采购单"** 按钮
4. 系统提示："采购单 PO20260609XXX 已创建，已流转至仓库待入库"


## 💡 使用技巧

### 技巧1：批量添加同类物品

```
推荐做法：
1. 先连续点击"+"添加所有需要的行数
2. 然后从上到下依次填写
3. 相似物品可以连续添加，利用历史记录

例如采购文具：
  第1行：中性笔 - 晨光 - 0.5mm
  第2行：中性笔 - 晨光 - 0.7mm（型号不同）
  第3行：中性笔 - 得力 - 0.5mm（品牌不同）

```

## 🔍 常见问题速查

### Q: 如何删除不需要的物品行？
**A:** 点击该行最右侧的 **✕** 按钮

### Q: 类别输入后没有出现在下拉列表？
**A:** 确保输入后点击了其他地方（失焦），或刷新页面

### Q: 品牌型号历史如何清除？
**A:** 目前不支持手动清除，会永久保存。如需清理可在浏览器控制台执行清除命令

### Q: 商品编码可以修改吗？
**A:** 不可以，SKU编码由系统自动生成以保证唯一性

### Q: 金额计算不准确？
**A:** 检查数量和单价是否正确输入，金额 = 数量 × 单价

### Q: 刷新页面后数据丢失？
**A:** 提交前的采购单不会保存，提交后会永久保存到localStorage
