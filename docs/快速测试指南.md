# 采购入库功能快速测试指南

## 🚀 快速开始

### 1. 打开系统
双击 `打开库存管理系统.bat` 或直接在浏览器中打开 `index.html`

## 🧪 测试场景一：手动创建采购单

### 步骤：
1. **登录采购员账号**
   - 用户名：`purchase`
   - 密码：`purchase123`

2. **进入采购单管理**
   - 点击左侧菜单"业务流程" → "采购单管理"

3. **新建采购单**
   - 点击"+ 新建采购单"按钮
   - 填写信息：
     ```
     采购日期：2026-06-09（默认）
     采购人：张三
     供应商：XX食品供应商
     ```

4. **添加物品明细**
   - 点击"+ 添加物品"
   - 填写第一个物品：
     ```
     物品名称：矿泉水
     品牌：农夫山泉
     型号：550ml
     数量：100
     单位：瓶
     单价：2.5
     ```
   - 再次点击"+ 添加物品"
   - 填写第二个物品：
     ```
     物品名称：方便面
     品牌：康师傅
     型号：桶装
     数量：50
     单位：箱
     单价：45
     ```

5. **提交采购单**
   - 查看合计金额是否正确（应为 2500.00）
   - 点击"提交采购单"
   - 提示："采购单 PO20260609XXX 已创建，已流转至仓库待入库"

6. **验证结果**
   - 采购单列表中出现新记录
   - 状态显示为"待入库"（黄色徽章）
   - 出现"入库"按钮（但采购员看不到，需要切换账号）

## 🧪 测试场景二：Excel导入采购单

### 步骤：
1. **下载模板**
   - 在采购单管理页面
   - 点击"📄 导入Excel"按钮
   - 点击"📥 下载Excel模板"
   - 保存文件到本地

2. **编辑Excel**
   - 打开下载的模板文件
   - 修改示例数据或添加新行
   - 保存文件

3. **上传导入**
   - 回到系统，仍在导入模态框
   - 点击"下一步"进入步骤2
   - 点击上传区域，选择刚才的Excel文件
   - 或直接拖拽文件到上传区域

4. **预览数据**
   - 系统自动解析并显示预览表格
   - 检查数据是否正确
   - 查看统计信息（记录数、总金额）

5. **确认导入**
   - 点击"下一步"直到步骤4
   - 点击"确认导入"
   - 提示："成功导入 X 个采购单"

6. **验证结果**
   - 关闭模态框
   - 采购单列表显示新导入的采购单
   - 状态为"待入库"

---

## 🧪 测试场景三：仓库确认入库

### 步骤：
1. **切换到仓库管理员账号**
   - 点击右上角"退出"按钮
   - 重新登录：
     ```
     用户名：warehouse
     密码：warehouse123
     ```

2. **查看待入库采购单**
   - 进入"采购单管理"
   - 看到之前创建的采购单
   - 状态为"待入库"

3. **确认入库**
   - 点击"入库"按钮
   - 弹出"入库确认"模态框
   - 查看采购单信息是否正确

4. **核对入库明细**
   - 检查每个物品的信息
   - 修改"实收数量"（如有差异）
   - 例如：矿泉水实收98瓶（有2瓶破损）

5. **填写批次号和备注**
   - 批次号：自动生成，可修改
   - 验收备注：如实记录"2瓶破损，拒收"

6. **确认入库**
   - 点击"确认入库"按钮
   - 提示："入库成功！批次号：BATCHXXX\n已自动生成库存明细"

7. **验证结果**
   - 采购单状态变为"已入库"（绿色徽章）
   - "入库"按钮消失

## 🧪 测试场景四：查看库存明细

1. **进入库存概览**
   - 点击左侧菜单"库存管理" → "库存概览"

2. **查看自动生成的库存**
   - 应该能看到刚才入库的物品
   - 矿泉水：库存 98 瓶
   - 方便面：库存 50 箱

3. **验证数据来源**
   - 物品应标记为来自采购（source: "purchase"）
   - 显示最后入库日期和批次号

4. **进入入库管理**
   - 点击"业务流程" → "入库管理"
   - 查看入库记录列表
   - 点击"查看详情"查看完整信息

## 🧪 测试场景七：数据追溯测试

### 步骤：

   - 找到一个物品，记录其信息

4. **验证完整性**
   - 采购单 → 入库记录 → 库存明细

## ⚠️ 常见问题排查

### 问题1：点击按钮没反应
**检查：**
- 浏览器控制台是否有错误（F12）
- 是否正确加载了所有JS文件
- 是否使用了现代浏览器（Chrome/Edge/Firefox）

### 问题2：Excel导入失败
**检查：**
- Excel文件格式是否正确（.xlsx 或 .xls）
- 是否安装了 SheetJS 库（查看HTML中是否引入）
- 数据列是否符合模板格式

### 问题3：库存没有更新
**检查：**
- 是否完成了入库确认步骤
- 实收数量是否大于0
- 浏览器 localStorage 是否正常

### 问题4：数据丢失
**说明：**
- 当前版本使用 localStorage 存储
- 清除浏览器缓存会丢失数据
- 建议定期导出数据备份
